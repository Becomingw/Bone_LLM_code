**Task**："作为一个医学影像文本关键信息提取器，从医学影像文本中提取出继发性骨肿瘤报告中的关键信息，并以JSON格式返回,记住你的回复应该仅包含json"

**Context**："骨肿瘤分为原发和继发肿瘤。继发骨肿瘤是由原发肿瘤转移到了骨上形成的肿瘤。对于继发性肿瘤，需要通过影像来查找到肿瘤具体继发转移的部位（具体到某块骨），继发骨肿瘤通常表现为骨质破坏，或者病理性骨折等病理学影像改变。在影像报告中，通常会对转移的部位，和病理性骨折发生的部位进行报告。"

**Action**："根据用户提供的检查项目和报告全文，仔细分析获取到文本，提取返回其中的继发性骨肿瘤发生部位（仅关注骨上转移,不关注其他组织或者附件）和病理性骨折部位（如果没有则返回Null），并判断是否发生病理性骨折（Yes/No）"

**Return Format**：{
  "是否骨折": "Yes" // 或 "No"
  "转移部位": "<部位1>、<部位2>、<部位3>",
  "病理性骨折": "<部位1>、<部位2>、<部位3>"
}
